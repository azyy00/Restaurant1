# Menu & site photos

Drop the real photos into this folder (and one into `public/`) using these **exact** filenames.
They load automatically. Any missing file shows a branded "photo coming soon" tile instead of breaking.

Recommended: JPG, roughly **4:3** (e.g. 1200×900) for menu items. Keep each under ~300 KB.
The story photo is **4:5** portrait (e.g. 1000×1250).
